How this maps to the build

  • Inline progress mirrors the real concise/detailed tiers (clarifier → search → rerank → validator → finalizer). Toggle Detailed view in the top-right of the app to reveal agent names + counts; debug stays admin-only.
  • Streaming is replay of the validated answer (no second LLM call), matching R4-4. Citations open the document viewer with the quoted span highlighted; Law says vs Your document says are visually distinct.
  • Uploads show the processing pipeline (Uploaded → Parsed → Chunking → Embedding → Ready); files land in the library and are explicitly attached to a chat.
  • Locked tokens live in docs/design/tokens.json → CSS variables → Flutter ThemeData. Hindi uses Noto Sans Devanagari with a size + leading boost.
