What changed (and why dark feels better now)

  • Elevation, not tint — each surface steps up in lightness (bg → surface → raised → hover) so cards read as lifted, not colored. Borders are white-alpha hairlines + a inset 0 1px 0 top highlight.
  • Calmer accent — one primary hue per theme; citations use neutral body text with only the label colored, so it never looks muddy.
  • Glow control — Off / Soft / Glow. Soft is the tasteful default; Glow is for demos/marketing shots.
  • Type size — stage text is em-based off a --ts multiplier; XL targets 4K. On mobile the same scale keeps ≥16px tap-friendly text.
  • Hindi — toggle swaps UI strings and switches the stack to Noto Sans/Serif Devanagari. Core layout never hardcodes English.
