Flutter / React portability

Each variant maps to data-variant keys in tokens.json. React: CSS variables on :root + Tailwind theme extension. Flutter: ThemeData + TextTheme from the same JSON; use MediaQuery.textScaler clamp for 4K; VisualDensity.compact for dense variant. Motion: AnimationController(220ms, Curves.easeOutCubic) matching --dur-normal.
